Page de connexion au Mumble de la FK

  • Ouvrez Mumble
  • Cliquez sur "Serveur" dans la barre de menus
  • Cliquez sur "Se connecter..."
  • Dans la boîte de dialogue "Se connecter au serveur", entrez les informations suivantes :
    • Nom : FKTeam Mumble
    • Adresse : fkteam.mumble.gg
    • Port : 10003
    • Nom d'utilisateur : [Votre pseudo]
